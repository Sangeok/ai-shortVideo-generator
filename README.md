## üì¶ Installation

### Prerequisites

- Node.js 18.0 or higher
- npm or yarn
- OpenAI API key

- Deepgram API key

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/ai-short-video-generator.git
cd ai-short-video-generator
```

### 2. Install Dependencies

```bash
npm install
# or

```

### 3. Run Development Server

```bash

# or

```
